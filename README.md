# A Little About Me. 

## Description
This is my personal website where I showcase my portfolio, blog about web development, and share my resume. It's designed to highlight my skills and projects.

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Installation Instructions
1. Clone the repository: `https://github.com/Cynmcc8799/Cynmcc8799.github.io.git`
2. Open the `index.html` file in your web browser.

## Usage
Navigate through the website using the menu at the top. You can view my projects, read my blog posts, or download my resume.
